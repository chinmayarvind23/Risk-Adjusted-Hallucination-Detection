# Risk-Adjusted-Hallucination-Detection

This repository contains code for a risk-adjusted hallucination detector for grounded QA using uncertainty and evidence-based features, calibrates hallucination risk, and studies abstention and cross-regime transfer between PHANTOM and a retrieval-grounded HalluLens subset.

PHANTOM dataset generations can be found [here](https://drive.google.com/drive/folders/1zdo_osGhLjunwaIf5vK4wXX-ptWU83zk)
